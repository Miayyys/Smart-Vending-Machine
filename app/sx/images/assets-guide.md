# 图片资源说明

## 项目需要的图片文件

### 首页横幅
- `banner.jpg` - 首页横幅图片（建议 750x400 像素）

### 默认图标
- `product-default.png` - 商品默认图片（建议 200x200 像素）
- `avatar-default.png` - 默认头像（建议 200x200 像素）

### TabBar 图标（可选）
当您准备好图标后，请添加到 images 目录：
- `home.png` / `home-active.png` - 首页图标
- `products.png` / `products-active.png` - 商品图标
- `orders.png` / `orders-active.png` - 订单图标
- `profile.png` / `profile-active.png` - 我的图标

## 临时解决方案

如果您暂时没有这些图片，可以使用以下方法：

1. **纯色背景**：暂时使用纯色背景
2. **文字代替**：使用文字显示
3. **使用 base64 图片**：内联小图片

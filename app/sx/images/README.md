# 图标说明

## TabBar 图标要求

1. **尺寸**: 81px × 81px
2. **格式**: PNG 格式，透明背景
3. **风格**: 简洁、清晰的线条图标

## 图标清单

### 首页 (home.png / home-active.png)
- 未选中：灰色线条图标
- 选中：绿色线条图标

### 商品 (products.png / products-active.png)
- 未选中：灰色线条图标
- 选中：绿色线条图标

### 订单 (orders.png / orders-active.png)
- 未选中：灰色线条图标
- 选中：绿色线条图标

### 我的 (profile.png / profile-active.png)
- 未选中：灰色线条图标
- 选中：绿色线条图标

## 获取图标来源

1. **图标库**: 
   - [Iconfont](https://www.iconfont.cn/)
   - [Flaticon](https://www.flaticon.com/)
   - [Material Icons](https://material.io/resources/icons/)

2. **设计工具**:
   - Figma
   - Sketch
   - Adobe XD

## 使用示例

从图标库下载后，将文件重命名为对应的名称，放入 `images/` 目录即可。
